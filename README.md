# ProjectNew

Root project (monorepo) setup for ProjectNew.

## Prerequisites

- Node.js 20+
- pnpm 9+

## Quick start

```bash
pnpm install
pnpm dev
```

## Common scripts

- `pnpm dev` - run all workspace dev tasks
- `pnpm build` - build all packages/apps
- `pnpm test` - run tests across workspaces
- `pnpm lint` - run lint tasks
- `pnpm format` - run formatting tasks
